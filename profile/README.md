## About

OpenHospi is a free, open-source student housing platform built for the Netherlands. Students verify their identity through **[InAcademia](https://inacademia.org/)** — so you know you're dealing with real students — and all messages are **end-to-end encrypted**. No paywalls, no middlemen, fully transparent.

---

## Key Features

| | Feature |
|---|---|
| **Verified** | InAcademia student verification — only real students |
| **Private** | End-to-end encrypted messaging |
| **Free** | 100% free, no premium tiers or hidden costs |
| **Open** | Open source and community-driven |
| **Dutch** | Built around the traditional "hospiteren" process |

---

## Tech Stack

| Layer | Technology |
|---|---|
| Web | Next.js |
| Mobile | Expo (React Native) |
| Database | Supabase + Drizzle ORM |
| Storage | Supabase Storage |
| Auth | Better Auth + InAcademia |
| Monorepo | pnpm workspaces |
